위 파일들을 기준으로 이 포트폴리오를 분석해줘.

목표
- 채용 담당자가 첫 30초 안에 강점을 이해할 수 있는지 본다
- 실무 면접관이 추가 질문하고 싶어질 정도로 기술적 깊이가 보이는지 본다
- 테크 리드 관점에서 설계 판단, 기여도, 전달력이 보이는지 본다

출력 형식
1. 한 줄 총평
2. 좋은 점 3개
3. 문제점 3개
4. 누락된 핵심 요소
5. 바로 고칠 문장 예시
6. 다음 액션 3개

저장 규칙
- 결과는 반드시 $1 파일에 저장한다
- $1은 docs/analysis/portfolio/ 아래 경로여야 한다
- 저장 파일 상단에는 아래 메타 정보를 포함한다
  - 분석일
  - 분석 대상
  - 분석 종류: portfolio-review

주의사항
- 실제 소스 파일은 수정하지 않는다
- 근거 없는 성과 수치나 과장 표현은 만들지 않는다
- 가능한 한 어떤 파일의 어떤 부분이 약한지 구체적으로 짚는다
