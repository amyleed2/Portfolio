# 포트폴리오 감사 Skill

이 Skill은 포트폴리오를 수정하기 전에 현재 상태를 진단할 때 사용한다.

## 목적

현재 포트폴리오가 10년 이상 경력의 Senior iOS Application Developer를 채용 관점에서 잘 보여주는지 확인한다.

## 실행 단계

1. 다음 파일을 읽는다.

- index.html
- styles.css
- script.js
- career-modal.js
- project-modal.js
- docs/portfolio-renewal/profile-source.md
- docs/portfolio-renewal/strategy.md

2. 다음 세 관점에서 평가한다.

- 채용 담당자
- iOS 현업 면접관
- 개발팀 리드

3. 다음 항목을 확인한다.

- 첫 화면에서 5초 안에 Senior iOS 포지셔닝이 보이는가?
- 경력 정보가 한눈에 들어오는가?
- 프로젝트가 회사별 경력과 자연스럽게 연결되는가?
- 프로젝트 설명이 충분한가?
- 프로젝트 설명이 너무 길거나 장황하지 않은가?
- 오래된 프로젝트가 현재 강점을 흐리지 않는가?
- AI 활용 경험이 구체적이고 믿을 만한가?
- 사이드 프로젝트가 실무 역량을 보완하는가?
- 시각적 정보 위계가 명확한가?
- 모바일 화면에서도 읽기 쉬울 가능성이 있는가?
