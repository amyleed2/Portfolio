## AI 활용 전략

AI는 별도 섹션으로 분리한다.

AI 활용을 단순 도구 나열이 아니라 다음 방식으로 보여준다.

- 코드 분석
- 구조 설계 초안
- 테스트 케이스 작성
- 리팩토링 검토
- 문서화
- 사이드 프로젝트 기획
- UI 콘셉트 탐색

중요한 문장:

AI를 단순 코드 생성 도구가 아니라, 설계 검토·리팩토링·테스트·문서화의 생산성을 높이는 보조 도구로 활용합니다.  
최종 설계 판단과 코드 품질 검증은 개발자가 수행합니다.
